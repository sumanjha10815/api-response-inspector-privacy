# API Response Inspector – Privacy Policy

This repository contains the official Privacy Policy for the **API Response Inspector** Chrome DevTools extension.

## Purpose

The API Response Inspector extension allows developers to inspect, analyze, and export API network requests and responses directly within Chrome DevTools.

This repository exists to:
- Publicly host the extension’s Privacy Policy
- Comply with Chrome Web Store Developer Program requirements
- Provide a transparent explanation of data handling practices

## Hosted Privacy Policy

The Privacy Policy is published via **GitHub Pages** and is accessible at:

> https://sumanjha10815.github.io/api-response-inspector-privacy/

## Data Handling Summary

- All data is processed locally in the user’s browser
- No data is transmitted to external servers
- No analytics, tracking, or third-party services are used
- Users retain full control over captured data
